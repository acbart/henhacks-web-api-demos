Source Code

// Make an element draggable
<div draggable="true" id="card">Drag me!</div>

// Drag source events
card.addEventListener('dragstart', (e) => {
  e.dataTransfer.setData('text/plain', card.id);
  e.dataTransfer.effectAllowed = 'move';
});

// Drop target events
dropTarget.addEventListener('dragover', (e) => {
  e.preventDefault(); // required â€” otherwise drop is not allowed
  e.dataTransfer.dropEffect = 'move';
  dropTarget.classList.add('drag-over');
});

dropTarget.addEventListener('drop', (e) => {
  e.preventDefault();
  const id = e.dataTransfer.getData('text/plain');
  const draggedEl = document.getElementById(id);
  dropTarget.appendChild(draggedEl);
});

// Accept file drops from the desktop
dropZone.addEventListener('drop', (e) => {
  e.preventDefault();
  const files = e.dataTransfer.files; // FileList
  for (const file of files) {
    console.log(file.name, file.size, file.type);
  }
});

Key Facts

  • Set draggable="true" on any element to make it draggable.
  • You must call event.preventDefault() in the dragover handler to allow dropping.
  • Use dataTransfer.setData()/getData() to pass data between the drag source and drop target.
  • File drops from the OS are accessed via event.dataTransfer.files â€” a FileList.
  • dataTransfer.items gives you the dragged items as a DataTransferItemList for more control.
  • Touch devices don't support the native DnD API â€” use Pointer Events for cross-device drag-and-drop.
