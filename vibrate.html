Source Code

// Single vibration (milliseconds)
navigator.vibrate(200);      // vibrate for 200ms

// Pattern: [vibrate, pause, vibrate, pause, ...]
navigator.vibrate([200, 100, 200]);         // two short pulses
navigator.vibrate([100, 50, 100, 50, 500]); // SOS-like pattern

// Stop any ongoing vibration
navigator.vibrate(0);
navigator.vibrate([]);  // equivalent

// Check support
if ('vibrate' in navigator) {
  console.log('Vibration API supported');
} else {
  console.log('Not supported ‚Äî desktop or iOS');
}

// Example: tactile button feedback
button.addEventListener('click', () => {
  navigator.vibrate(30); // subtle 30ms click feel
});

// Notification pattern
function notifyUser() {
  navigator.vibrate([300, 200, 300, 200, 300]);
}

Key Facts

  • One of the simplest browser APIs ‚Äî just navigator.vibrate(pattern).
  • The pattern array alternates between vibrate and pause durations in milliseconds.
  • Supported on Android Chrome and Firefox. Not supported on iOS (Apple's policy).
  • Requires a user gesture (e.g., a click) before the page can vibrate ‚Äî prevents annoying auto-vibrations.
  • Use short (20‚Äì50ms) pulses for subtle feedback, longer for alerts, and patterns for notifications.
  • Great for games (hit feedback), form validation errors, and accessibility cues.
